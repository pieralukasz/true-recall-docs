Quick reference for all True Recall keyboard shortcuts.

## During Review

### Answer Rating

| Key | Action |
|-----|--------|
| `1` | Rate: Again |
| `2` | Rate: Hard |
| `3` or `Space` | Rate: Good |
| `4` | Rate: Easy |

### Showing Answer

| Key | Action |
|-----|--------|
| `Space` | Show answer (when hidden) |
| `Cmd/Ctrl + Enter` | Submit typed answer (type-in mode) |

### Card Actions

| Key | Action |
|-----|--------|
| `E` | Edit card |
| `!` (Shift+1) | Suspend card |
| `-` | Bury card until tomorrow |
| `=` | Bury note (all siblings) until tomorrow |
| `M` | Move card to another note |
| `A` | Add new flashcard |
| `T` | Cycle type-in mode (Off -> AI -> Diff -> Off) |
| `P` | Set source note preset (`fsrs_preset`) |

### Session Actions

| Key | Action |
|-----|--------|
| `Cmd/Ctrl + Z` | Undo last answer |
| `Escape` | End session |
| `?` | Show keyboard shortcuts help |

## [Card Browser](/views/card-browser/)

| Key | Action |
|-----|--------|
| `j` / Down arrow | Move down |
| `k` / Up arrow | Move up |
| `Enter` | Edit selected card |
| `Space` | Toggle selection |
| `Cmd/Ctrl + A` | Select all |
| `Escape` | Clear selection |

## [Flashcard Panel](/views/flashcard-panel/)

| Key | Action |
|-----|--------|
| `Enter` | Edit selected card |
| `Delete` | Delete selected card |
| `Escape` | Deselect |

| Command | Default Shortcut |
|---------|-----------------|
| Open dashboard | -- |
| Open flashcard panel | -- |
| Review flashcards from current note | -- |
| Review today's new cards | -- |
| Open card browser | -- |
| Open statistics panel | -- |

### Setting Custom Shortcuts

1. Open Obsidian Settings
2. Go to Community Plugins
3. Click keyboard icon next to True Recall
4. Set your preferred shortcuts

### Recommended Shortcuts

| Command | Suggested Shortcut |
|---------|-------------------|
| Review current note | `Cmd/Ctrl + R` |
| Open dashboard | `Cmd/Ctrl + D` |
