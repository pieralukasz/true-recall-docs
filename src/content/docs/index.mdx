## Install in 2 Minutes
</div>

<Steps>
1. **Download Obsidian** — Get it free from [obsidian.md](https://obsidian.md) and create a vault

2. **Install True Recall** — Open `Settings → Community plugins → Browse`, search for **True Recall**, and click **Install**

3. **Enable True Recall** — Toggle it on in `Settings → Community plugins`

4. **Start creating cards** — Select text in any note, click the ✨ button, and AI generates flashcards instantly
</Steps>

Available directly in the [Obsidian Community Plugins](https://community.obsidian.md/plugins/true-recall) directory — no BRAT or manual install needed.
